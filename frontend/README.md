# Description

Below are instructions on how to run the project locally. Since these are
generated files they will change overtime as more work is being done.

## Getting Started

- Install [pnpm](https://pnpm.io/installation#on-windows)
- Then run`pnpm install` in this folder
- Run `pnpm dev` which should be running on
  [http://localhost:3000](http://localhost:3000)
